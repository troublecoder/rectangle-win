# Rectangle Win — 프론트엔드 구현 계획

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development.

**Goal:** Vue 3 + Nuxt UI + vue-konva로 Tauri 데스크톱 앱 프론트엔드를 구축한다.

**Architecture:** Vue SPA (Nuxt 프레임워크 아님). Vite + Nuxt UI (Vue 모드). FSD 경량화 구조. Tauri `invoke`로 백엔드 command 호출.

**Tech Stack:** Vue 3, Vite, @nuxt/ui (Vue mode), vue-konva, @tauri-apps/api, tailwindcss.

- [ ] **Step 4: src/features/config/useConfig.ts**

```typescript
import { ref, watch } from 'vue'
import type { Config } from '@/entities/config'
import { getConfig, saveConfig } from './api'

const config = ref<Config | null>(null)
const loading = ref(true)
const saving = ref(false)

export function useConfig() {
  async function load() {
    loading.value = true
    try {
      config.value = await getConfig()
    } finally {
      loading.value = false
    }
  }

  async function save() {
    if (!config.value) return
    saving.value = true
    try {
      await saveConfig(config.value)
    } finally {
      saving.value = false
    }
  }

  // debounce save — config 변경시 300ms 후 자동 저장
  let saveTimer: ReturnType<typeof setTimeout> | null = null
  watch(config, () => {
    if (saveTimer) clearTimeout(saveTimer)
    saveTimer = setTimeout(() => save(), 300)
  }, { deep: true })

  return { config, loading, saving, load, save }
}
```
